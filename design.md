## 0. 로고·엠블럼 사용 원칙 (중요)

화면 좌상단 마크는 **무궁화 꽃잎이 없는 파랑+빨강 태극 소용돌이** — 2016년 제정된
**현행 대한민국정부 공식 상징(정부상징)**이다. 노사누리 전용 로고가 아니다.

- **정부상징(태극 엠블럼)**: 절대 스크린샷에서 크롭하지 말 것. 관공서 문서에서 흐릿하면 바로 티가 난다.
  공식 벡터(AI/SVG/PNG)를 `gov.kr` 정부상징 안내에서 무료로 받아 **원본 벡터**로 사용한다.
- **워드마크 "노사누리 · 근로감독행정시스템"**: 공식 벡터가 없다. 시스템 자산 파일이 있으면 그것을,
  없으면 캡처에서 크롭하거나 동일 서체·색으로 재제작한다.
- 이 폴더에 PPT용 임시 크롭을 만들어 둠:
  - `assets_emblem_only.png` — 태극 엠블럼만 (임시용, 가능하면 공식 벡터로 교체)
  - `assets_logo_lockup.png` — 엠블럼 + 워드마크 가로 조합

## 1. 브랜드 정체성

- **주체**: 고용노동부(MOEL) — 노사관계·근로감독 업무 포털 "노사누리 / 근로감독행정시스템".
- **핵심 인상**: 공공기관의 **신뢰·공정·안정**. 화려함보다 명료함, 정보 위계가 분명한 업무형(admin) UI.
- **키워드**: 신뢰(Trust) · 공정(Fairness) · 안정(Stability) · 정직한 노동(Labor).
- 화면 전반이 **네이비 헤더 + 밝은 블루 강조 + 화이트 카드** 3단 구조. 여기에 정부 상단 **레드 라인**이 시그니처.

## 3. 타이포그래피

- **폰트**: 맑은 고딕(Malgun Gothic) — 화면 UI와 동일 계열, 한글 안전.
- **위계**
  - 대제목/키워드: 40~54pt, Bold
  - 카드 제목/섹션 헤더: 16~20pt, Bold
  - 소제목/탭 라벨: 14~16pt, Bold
  - 본문/정의: 18~22pt, Regular/Semibold
  - 캡션/각주/날짜: 11~13pt, Regular, 그레이
- 강조는 **색·굵기**로만. 밑줄·이탤릭 남용 금지. 숫자 카운트는 굵게 + 컬러 뱃지.

## 4. 레이아웃 원칙 (화면 관찰)

- **최상단 레드 라인**: 얇은 정부 레드 풀-폭 라인 → 관공서 시그니처.
- **상단 네이비 헤더 바**: 좌측 정부상징+워드마크, 중앙 가로 메뉴, 우측 사용자/타이머/로그아웃.
- **좌측 아이콘 레일**: 라이트 그레이 배경 + 세로 아이콘 메뉴(알림/결재/SMS 등).
- **카드 그리드**: 화이트 카드 + 얇은 블루 테두리 + 라운드 코너. "나의 업무/기능 업무/결재 대기" 식 분할.
- **카운트 뱃지**: 파란 알약형 뱃지에 숫자, 위험은 레드·기한은 골드로 상태 색분리.
- **넉넉한 여백**과 좌측 정렬 위계(라벨 → 제목 → 값)로 신뢰감 있는 공백 유지.
- **하단/우측 보조 위젯**: 캘린더, 배너, 사용문의 등 라이트 톤 카드로 정리.

## 5. 아이콘·컴포넌트 톤

- 아이콘은 **면(fill) + 라인 혼합**, 네이비/블루 단색. 방패(인증)·트로피(명예의전당) 등 상징적 일러스트는 블루 그라데이션.
- 버튼: 네이비 솔리드(주요 액션) / 화이트+네이비 테두리(보조). 라운드 4~6px.
- 링크·포인트 텍스트는 시그널 블루, 위험 문구는 레드.

## 6. 슬라이드(PPT) 적용 가이드

- 16:9 화이트 배경 + 상단 네이비 헤더 바 + 최상단 얇은 레드 라인.
- 좌상단에 **정부상징(공식 벡터) + 워드마크** 조합 배치.
- 대형 키워드는 Bold, 따뜻한 강조는 어닝 골드(`#FFC000`) 절제 사용.
- 근거 3요소는 화이트/라이트블루 카드로 3분할, 상태값은 블루/레드/골드 뱃지로 시각화.
- 전체적으로 절제된 색·명확한 위계·공공 신뢰 톤 유지.
